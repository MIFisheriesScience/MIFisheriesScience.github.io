Most fisheries papers include a map. This week we'll learn how to build maps in R.

### Lecture Topics

* Mapping basics
	+ Projections
	+ Coordinates
* Using the marmap package	
* Comparison with other mapping packages
* ggplot2 maps
* Adding an inset
* Plotting NAFO zones
